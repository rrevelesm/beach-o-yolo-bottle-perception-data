# Sample Preview

This folder contains a **10-image lightweight preview** of the dataset — 8 positive (bottle-present) and 2 negative (empty-label) examples from the training split — so the folder structure, image format, and label format can be inspected without downloading the full ~2 GB dataset.

This is **not** a representative statistical sample and must not be used for training or evaluation. Use `../dataset_yolo/` (full dataset, via the public OneDrive link in `../README_DATASET.md`) for any actual training or benchmarking.

| File | Type |
|---|---|
| `image_0001`–`image_0008` | positive (bottle present) |
| `image_0266`, `image_0305` | negative (empty label file) |
