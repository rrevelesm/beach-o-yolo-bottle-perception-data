# Release Notes — v1.0

## Contents

- `dataset_yolo/` — final sequence-aware, leakage-free YOLO dataset (1585 images / 1585 labels / 3167 boxes; 1267 train / 318 validation).
- `bags/20241128_133505.bag` — RealSense RGB recording used for offline BAG replay.
- `metadata/` — dataset summary, sequence-split summary, class mapping, acquisition notes, label audit report.
- `checksums/` — SHA256 manifests for the dataset and the BAG file, plus a full file inventory.
- `sample_preview/` — a 10-image lightweight preview subset (not the full dataset).

## Corresponds to

The dataset and BAG recording behind the results reported in the Robotics manuscript "YOLO-Based Visual Perception for Bottle Detection in a Beach-Cleaning Robot Using RealSense BAG Sequences and Embedded Deployment Analysis."
